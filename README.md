# Incident Response Playbook (SOC-Focused)

This repository contains practical incident response playbooks and templates designed to mirror Tier 1 SOC workflows. The goal is to demonstrate structured triage, validation, excalation, and documentation processes.

## Contents
- Brute Force / Credential Stuffing Playbook
- Incident Ticket Template
- Additional playbooks in progress

## Intended Audience
- Entry-level SOC analysts
- Blue team learners
- Security operations trainees
